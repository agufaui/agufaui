# Agufa UI

## Installation:

```bash
npm install @agufaui/use
```
```bash
yarn add @agufaui/use
```
```bash
pnpm add @agufaui/use
```

### Vue 3

in any source file

```
import { aUseComponent } from '@agufaui/use'

const { getColorCSS } = aUseComponent()
```

### Nuxt 3

in any source file

```
import { aUseComponent } from '@agufaui/use'

const { getColorCSS } = aUseComponent()
```
